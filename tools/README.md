# Tools

This directory contains utility scripts and tools for the project.

## Setup Scripts

The `setup/` directory contains scripts for initial setup and configuration:

- **[setup.sh](setup/setup.sh)** - Linux/Unix setup script
- **[RUN_ME.bat](setup/RUN_ME.bat)** - Windows batch script for running the project
- **[check_and_setup_git.bat](setup/check_and_setup_git.bat)** - Windows Git setup script
- **[upload_to_github.ps1](setup/upload_to_github.ps1)** - PowerShell script for uploading to GitHub

## Usage

### Linux/Unix
```bash
cd tools/setup
./setup.sh
```

### Windows
```cmd
cd tools\setup
RUN_ME.bat
```

## Note

These scripts are provided for convenience. Refer to the [documentation](../docs/) for detailed instructions on using these tools.
